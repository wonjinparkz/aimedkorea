# Google Play Store 반려 요인 분석 및 체크리스트

## 🚫 주요 반려 사유 및 현재 프로젝트 상태

### 1. ❌ **의료/건강 관련 정책 위반** [주의 필요]

#### 반려 사유:
- 의학적 진단, 치료, 처방 제공
- COVID-19 관련 잘못된 정보
- 승인되지 않은 의료 기기 기능
- 의사 상담 대체 표현

#### 현재 프로젝트 상태:
- ⚠️ **위험**: "건강 진단", "자가진단" 용어 사용
- ⚠️ **위험**: "12주 회복 프로그램" - 의료적 치료 암시
- ⚠️ **위험**: 설문 결과가 의학적 진단으로 오해 가능

#### 필요한 조치:
```
1. 모든 "진단" 용어를 "체크", "평가"로 변경
2. 면책 조항 추가: "의학적 조언이 아님"
3. "회복" → "웰니스", "건강 관리"로 변경
```

---

### 2. ✅ **개인정보 처리 정책** [준수]

#### 반려 사유:
- 개인정보처리방침 URL 누락/접근 불가
- 데이터 세이프티 섹션 불일치
- 민감한 정보 수집 시 명시 부족

#### 현재 프로젝트 상태:
- ✅ **준수**: 개인정보처리방침 페이지 존재
- ✅ **준수**: 데이터 세이프티 문서 준비 완료
- ✅ **준수**: 사용자 동의 절차 구현

---

### 3. ⚠️ **권한 사용 정당성** [부분 준수]

#### 반려 사유:
- 불필요한 권한 요청
- 권한 사용 설명 부족
- 민감한 권한의 부적절한 사용

#### 현재 프로젝트 상태:
- ✅ **준수**: 최소한의 권한만 요청
- ⚠️ **개선 필요**: 위치 권한 사용처 불명확
- ⚠️ **개선 필요**: 카메라 권한 실제 기능 미구현

#### 필요한 조치:
```
1. 미사용 권한 제거 (위치, 카메라)
2. 또는 해당 기능 실제 구현
3. 권한 요청 시 사용 목적 팝업 추가
```

### 4. ❌ **콘텐츠 등급** [미완료]

#### 반려 사유:

- 부적절한 연령 등급 설정

## 📋 즉시 수정 필요 사항

### 우선순위 1 (반려 위험 높음):
1. **의료 관련 용어 변경**
   - "진단" → "체크", "평가"
   - "회복" → "웰니스"
   - "치료" → "관리"

2. **면책 조항 추가**
   ```html
   <div class="disclaimer">
   본 서비스는 의학적 조언, 진단, 치료를 대체하지 않습니다.
   건강 상태에 대한 정확한 진단은 의료 전문가와 상담하세요.
   </div>
   ```

3. **연령 제한 설정**
   - 회원가입 시 13세 이상 확인
   - 약관에 연령 제한 명시

### 우선순위 2 (개선 권장):
1. **신고 기능 구현**
   - 댓글 신고 버튼
   - 관리자 알림 시스템

2. **권한 정리**
   - 미사용 권한 제거
   - 권한 요청 시 설명 팝업

3. **스크린샷 준비**
   - 실제 앱 스크린샷 5개 이상
   - 권한 요청 화면 포함
